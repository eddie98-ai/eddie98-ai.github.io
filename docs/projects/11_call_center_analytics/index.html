AI-Powered Call Center Analytics Platform

An end-to-end platform to transcribe, analyze, and visualize call center audio using LLMs and custom Deep Learning models.

Call center managers face a significant challenge in effectively analyzing the vast volume of audio calls to extract actionable insights. This project presents an intelligent, AI-driven analytics platform that fully automates the process, from converting audio files into structured transcripts to analyzing them with Large Language Models and culminating in an interactive API for querying results.

1. System Architecture & Pipeline

The system was designed as an integrated pipeline consisting of several logical phases, each built upon state-of-the-art technologies in Natural Language Processing (NLP) and Deep Learning.

Phase 1: Speech-to-Text with Speaker Diarization

The foundation of the project is the accurate conversion of audio into text. The advanced WhisperX model was chosen for this task due to its ability to provide high-accuracy transcription along with Speaker Diarization, which automatically identifies and separates the customer's speech from the agent's—a critical step for all subsequent analysis.

Phase 2: Text Analysis with LLMs

Once structured text was obtained, a suite of pre-trained Transformer models, served via a FastAPI application, was used to extract key features like Abstractive Summarization (facebook/bart-large-cnn), Sentiment Analysis (cardiffnlp/twitter-roberta-base-sentiment-latest), and initial Call Outcome Classification (Zero-Shot with facebook/bart-large-mnli).

2. Custom Deep Learning for Outcome Classification

To achieve superior accuracy in determining if a customer's problem was resolved, a custom solution was developed. A DistilBERT model was fine-tuned on a large, tailored dataset (AIxBlock/92k-real-world-call-center-scripts-english), learning the specific patterns of support dialogues to create a highly specialized and accurate classifier.

3. RAG-Powered Interactive Chatbot

To make the analyzed data queryable, an advanced chatbot was developed using a Retrieval-Augmented Generation (RAG) architecture. This chatbot searches the knowledge base of call summaries to find information relevant to a user's query and then uses the t5-base model to generate a natural, concise answer supported by sources.

Technical Summary

  • Speech-to-Text: WhisperX with Speaker Diarization
  • LLM Analysis: Transformers (BART, RoBERTa), Zero-Shot Classification
  • Custom Model: Fine-tuned DistilBERT with TensorFlow/Keras
  • Chatbot: Retrieval-Augmented Generation (RAG) with T5
  • Backend: FastAPI
